# PRODEMGE

- PHP 8.3
- Laravel 11

# Setup

```
git clone git@github.com:matsantosz/progemge.git
cd progemge
composer install
php artisan key:gen
php artisan migrate
php artisan db:seed
npm install
```

```
php artisan serve
npm run dev
```
